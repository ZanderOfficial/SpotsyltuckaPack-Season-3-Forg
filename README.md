# SpotsyltuckaPack
## Welcome
Welcome to SpotsyltuckaPack! This modpack will go along with our Season 3 Modded Minecraft Server.

### Warning
This pack has temperature mods which change how your player reacts to the world around them. It also adds water as a need along with hunger. These mods along with seasons will effect the needs of your player.

#### This pack includes:
- Create mods and addons.
- Sareen Seasons and Homeostatic for temperature.
- Minimal overworld biome changes.
- You will be able to create Valkyrian Skies ships in this pack.
- For rendering Distant Horizons, chunky, and a number of performance based mods.
- Minimap is Xaeros.
- There are new bosses, unique items, and new structures to find.
- As well as a number of QOL mods.
- You can also make wine! and so much more!
